 Titanic Survival Analysis
# Project Description
This project analyzes the Titanic dataset from Kaggle to understand passenger survival patterns.
#  source
Titanic dataset from Kaggle (train.csv)
# Tools Used
- Python
- Pandas
- Matplotlib
# Analysis Performed
- Survival analysis based on gender
- Survival analysis based on age groups
- Survival analysis based on passenger class
# Conclusion
The analysis shows that female passengers and first-class passengers had higher survival rates.
